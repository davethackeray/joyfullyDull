# PRODUCT REQUIREMENTS DOCUMENT (PRD)

## Project Name: Joyfully Dull Digital Porch

**Core Philosophy:** A web sanctuary for the constant purposeful struggle of being human. The UI/UX must feel homely, tactile, and completely detached from the urgency of modern digital capitalism. No pop-ups, no visual clutter, no auto-playing videos.

### 1. Technology Stack (Strict Constraints)

* **Framework:** Astro (Static Site Generation for maximum speed).
* **Styling:** Tailwind CSS (strictly using the custom color palette below).
* **Hosting/Deployment:** GitHub repository deployed via GitHub Actions to Cloudflare Pages.
* **Architecture:** Progressive Web App (PWA) with a Web App Manifest so users can add it to their mobile home screens.
* **Analytics:** Integrate Fathom Analytics or Plausible (cookie-free, privacy-first tracking). Absolutely no Google Analytics.

### 2. Design System & UI/UX Guidelines

* **Color Palette (Strict Custom Hex Codes Required):**
* Background: `Sourdough Crust` (Warm, golden-brown off-white).
* Accents: `Schwielowsee Sunrise` (Soft, pale early-morning yellow).
* Buttons/Cards: `Faded Denim` (Muted, comfortable mid-blue).
* Text: `Earl Grey` (Dark, soft grey—#2b2b2b—never pure black).


* **Typography:** W3C AAA accessible. Use a highly legible, rounded Serif for body text (resembling a public domain book) and a typewriter/handwritten font for headers.
* **The "Hygge" Feel:** Use soft rounded corners (`rounded-xl` in Tailwind), generous whitespace, and subtle CSS transitions (no jerky animations).

### 3. Core Features & Integrations

**A. The "Live Now" Engine (YouTube Data API v3)**

* **Requirement:** A dynamic "LIVE" indicator at the top of the UI that *only* appears when a stream is actively broadcasting.
* **Execution:** Create a Cloudflare Worker that pings the YouTube Data API (`search` endpoint with `eventType=live`) every 3 minutes to check the channel's status.
* **Action:** If live, the Worker passes a `true` state to the frontend, rendering a pulsing `Schwielowsee Sunrise` button that links directly to the active `videoId`.

**B. The Joyfully Dull Schedule & Calendar Connectors**

* **Requirement:** A weekly schedule UI that allows users to add streams to their personal calendars.
* **Execution:** Generate `.ics` files for Apple/Outlook and URL parameters for Google Calendar. Include: *Simplicitea*, *Walkshopping*, *Breaditation*, and *Pizza Cake*.

**C. The VIP "Guestbook" (Subscriber Data Capture)**

* **Requirement:** A free sign-up form for the *Morning Pages* VIP streams. We need qualitative data to build community, not just an email list.
* **Execution:** Build a lightweight form (route through Cloudflare Workers to a simple Supabase or Turso database).
* **Fields:** Name, Email, and one mandatory human question: *"What is the most joyfully dull part of your day?"* (This gives Dave ammunition to talk *with* the audience during the streams).

**D. The VIP Room (Morning Pages)**

* **Requirement:** A gated or hidden route (e.g., `/morning-pages`) accessible only to those who signed up, where embedded, unlisted YouTube streams of the daily journal sessions live.

### The Strategist's "Neat Touches" for Launch

To make sure this site feels incredibly unique the second someone lands on it, instruct the AI to build in these three Easter eggs:

1. **The "Take a Breath" Pre-loader:** Since Astro loads instantly, artificially delay the load of the homepage by exactly 1.5 seconds. During that time, show a simple text prompt: *"Take a deep breath. You are safe from the algorithm here."*
2. **Dynamic Greeting:** Have the hero section read the user's local time. If it's morning, "Good morning. The kettle is on." If it's evening, "Good evening. Time to log off soon."
3. **The "Analog" Toggle:** Put a small toggle switch in the footer. When clicked, it plays a quiet, looping, high-fidelity audio track of the wind off the Schwielowsee and boots crunching on gravel. It turns the site into an ambient soundboard.
